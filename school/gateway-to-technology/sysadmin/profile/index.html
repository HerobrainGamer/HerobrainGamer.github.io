What It Does

Well this script runs an startup of PowerShell when placed in the "C:\Users\$USERNAME\Documents\WindowsPowerShell\profile.ps1".
It can be used to setup custom functions with other .ps1 files or just writting it out, creating custom aliases for existing commands,
changing where PowerShell "Starts" in, printing out a witty welcome message to the user, and a bunch of other things.

Download This Script.

function home

}

{

}

{

{

}

#Create some alias that we can use
New-Alias -Name d -Value Get-ChildItem
New-Alias -Name rspooler -Value restartspooler
New-Alias -Name rspo -Value restartspooler

New-Alias -Name rannumg -Value randomnumbergame
